# Blood Bank Management System

A web-based Blood Bank Management System built with Python Flask and SQLite. This project allows users to manage blood donations, requests, and inventory seamlessly.

## Features
- **Admin Dashboard**: Manage inventory and track donations.
- **Donor Portal**: Register as a donor and log blood donations.
- **Recipient Portal**: Request blood and track request status.
- **Secure Authentication**: Role-based access control.

## Technology Stack
- **Backend**: Python, Flask, Flask-CORS
- **Database**: SQLite (No configuration required)
- **Deployment Ready**: Configured for Render and Gunicorn

## How to Run Locally
1. Clone this repository.
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application: `python app.py`
